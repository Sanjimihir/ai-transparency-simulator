# AI Transparency Simulator

An interactive Streamlit-based application that demonstrates **AI explainability and transparency**
for loan approval decisions using:
- Feature importance
- SHAP explanations
- Counterfactual reasoning

## Features
- Loan approval prediction (Random Forest)
- Basic & SHAP-based explanations
- Counterfactual suggestions
- User trust feedback collection

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- SHAP
- Pandas, NumPy, Matplotlib

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
